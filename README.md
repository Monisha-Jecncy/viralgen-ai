```bash

```

```bash
python -m venv venv
```

### Step 4: Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Mac/Linux

```bash
source venv/bin/activate
```

### Step 5: Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 6: Run Backend Server

```bash
python run.py
```

Backend runs at:

```text
http://127.0.0.1:8000
```

Open `index.html` in browser or run frontend server.

Frontend runs at:

```text
http://localhost:3000
```
