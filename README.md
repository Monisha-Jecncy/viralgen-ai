# 🚀 ViralGen AI - Multi-Modal Generative AI Advertisement Platform

ViralGen AI is a **Multi-Modal Generative AI Advertisement Platform** that helps users create **AI-powered advertisement content and visuals** automatically. The platform generates creative marketing text and promotional images from simple user prompts using advanced AI models.

## 📌 Project Overview

ViralGen AI is designed to simplify advertisement creation by using **Generative AI**. Users can enter a product idea or advertisement concept, and the system generates:

- ✨ Advertisement copy/content  
- 🎨 AI-generated promotional visuals  
- 📢 Marketing slogans and branding ideas  
- 🖼️ Multi-modal output (Text + Images)

The system supports different advertisement styles to create professional and engaging promotional content.



## 🎯 Features

✅ AI-powered advertisement generation  
✅ Multi-modal content generation (Text + Images)  
✅ GPT-4 Turbo integration for content creation  
✅ DALL·E 3 integration for image generation  
✅ Chat-based interactive UI  
✅ Advertisement style customization  
✅ Responsive and modern UI design  
✅ Search and chat history support  
✅ FastAPI backend integration

## ⚙️ Installation & Setup

### Step 1: Clone Repository

```bash
git clone https://github.com/yourusername/viralgen-ai.git
```

### Step 2: Navigate to Project Folder

```bash
cd viralgen-ai
```

### Step 3: Create Virtual Environment

```bash
python -m venv venv
```

### Step 4: Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Mac/Linux

```bash
source venv/bin/activate
```

### Step 5: Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 6: Run Backend Server

```bash
python run.py
```

Backend runs at:

```text
http://127.0.0.1:8000
```

### Step 7: Run Frontend

Open `index.html` in browser or run frontend server.

Frontend runs at:

```text
http://localhost:3000
```

5. Final advertisement output displayed to user
